Usage:
------

Just type SystemFonts at the command line, and the program prints
* Default GUI font,
* Font used for menus,
* Font used for window caption,
* Font used for small window caption and column headers,
* Font used for statusbar and tooltips.


Exit Codes:
-----------

0: No Error
1: Unable to get system fonts metric


Compiling:
----------

This program has been compiled successfully with VC++ 6.
